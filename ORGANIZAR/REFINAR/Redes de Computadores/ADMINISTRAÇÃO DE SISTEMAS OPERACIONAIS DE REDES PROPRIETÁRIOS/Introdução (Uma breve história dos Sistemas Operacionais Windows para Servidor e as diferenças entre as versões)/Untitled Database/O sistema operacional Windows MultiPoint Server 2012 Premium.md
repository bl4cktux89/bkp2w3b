Descrição: Dá suporte para vários usuários que acessam o mesmo computador host diretamente usando mouses, teclados e monitores separados. Limitado a dois soquetes, 4 TB de RAM e um máximo de 22 sessões. Dá suporte para algumas funções, incluindo as funções de Servidor DHCP e DNS, mas não para outras, como o AD DS, o AD CS e o AD FS. Compatível com ingresso em domínio.
